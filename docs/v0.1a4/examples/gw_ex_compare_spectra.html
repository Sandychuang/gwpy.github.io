GWpy Example: comparing a channel’s Spectrum between two times¶

Problem¶

I’m interested in comparing the amplitude spectrum of a channel between a known ‘good’ time - where the spectrum is what we expect it to be - and a known ‘bad’ time - where some excess noise appeared and the spectrum changed appreciably.

Solution¶

First, we define what the ‘good’ and ‘bad’ times are:

>>> goodtime = 1061800700
>>> badtime = 1061524816

and how long we want to search each one

>>> duration = 120

Then we can fetch the TimeSeries data for both times for the in-loop photodiode signal for the intensity stabilisation servo (ISS) of the pre-stabilised laser (PSL):

>>> from gwpy.timeseries import TimeSeries
>>> gooddata = TimeSeries.fetch('L1:PSL-ISS_PDB_OUT_DQ', goodtime, goodtime+duration)
>>> gooddata.name = 'Clean data'
>>> baddata = TimeSeries.fetch('L1:PSL-ISS_PDB_OUT_DQ', badtime, badtime+duration)
>>> baddata.name = 'Noisy data'

We can now calculate the amplitude spectral density (ASD) for each time using the asd() method,

>>> goodasd = gooddata.asd(4, 2)
>>> badasd = baddata.asd(4, 2)

and make a comparison plot:

>>> plot = badasd.plot()
>>> ax = plot.gca()
>>> ax.plot(goodasd)
>>> ax.set_xlim(10, 8000)
>>> ax.set_ylim(1e-6, 5e-4)
>>> plot.show()

The extra peak at 620 Hz is worrying, so we can zoom in around that frequency range to see what’s going on:

>>> ax.set_xscale('linear')
>>> ax.set_xlim(600, 640)
>>> ax.set_ylim(1e-6, 5e-4)
>>> plot.refresh()
