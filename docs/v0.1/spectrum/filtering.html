Filtering frequency-domain dataÂ¶

Frequency-domain data (a Spectrum or a Spectrogram) can be filtered using an arbitrary filter using the filter() methods of those objects:

Spectrum.filter(*filt, **kwargs) Apply the given filter to this Spectrum.
Spectrogram.filter(*filt, **kwargs) Apply the given filter to this Spectrogram.
