GWpy Example: plotting a StateVector¶

Problem¶

I would like to examine the state of the internal seismic isolation system supporting the Fabry-Perot mirror at the end of the Y-arm at LHO, in order to investigate a noise source.

Access to these data is restricted to the LIGO Scientific Collaboration and the Virgo Collaboration, but collaboration members can use the NDS2 service to download data.

Solution¶

First, we define the bitmask for the state-vector in question, simply a list of what each binary bit in the vector represents:

>>> bits = [
        'Summary state',
        'State 1 damped',
        'Stage 1 isolated',
        'Stage 2 damped',
        'Stage 2 isolated',
        'Master switch ON',
        'Stage 1 WatchDog OK',
        'Stage 2 WatchDog OK',
>>> ]

and can download the data for the relevant GPS interval as follows:

>>> from gwpy.timeseries import StateVector
>>> data = StateVector.fetch('L1:ISI-ETMX_ODC_CHANNEL_OUT_DQ', 'May 22 2014 14:00', 'May 22 15:00', bits=bits)

At this point it is useful to downsample the StateVector from the natural sampling frequency of 4096 Hz to 16 Hz, simply for ease of plotting:

>>> data = data.resample(16)

Now we can make a plot:

>>> plot = data.plot(add_label='inset')
>>> plot.set_title('LLO ETMX internal seismic isolation state')
>>> plot.add_bitmask('0b11101110')
>>> plot.show()

(Source code, png)


Here we see that the seismic isolation system moved in and out of its operating state, as a result of a large seismic transient shaking the vacuum chamber.
